# Code of Conduct

This project follows a simple standard: be respectful, constructive, and welcoming.

## Expected Behavior

- Use inclusive and considerate language.
- Be respectful of differing viewpoints and experiences.
- Give and receive feedback gracefully.
- Focus discussions on improving the repository.

## Unacceptable Behavior

- Harassment, intimidation, or discriminatory language.
- Personal attacks or sustained disruption.
- Publishing private information without permission.
- Other conduct that would reasonably be considered inappropriate in a professional setting.

## Enforcement

Maintainers may remove comments, close issues, or decline contributions that violate this code of conduct.
